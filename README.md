# sfclient
## Simple File Client

A local file server written in C for learning Inter Process Communications.
use [sfs](https://github.com/deviantfero/sfserver) to play with it.

![sfs/sfc demo](https://j.gifs.com/nrOzkl.gif)

```sh
# compile
make
# run
bin/sfc
```
